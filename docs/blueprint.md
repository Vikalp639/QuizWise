# **App Name**: QuizWise

## Core Features:

- Topic Selection: Allows users to select a topic for the quiz (e.g., Wellness, Tech Trends).
- MCQ Generation: AI generates 5 multiple-choice questions (MCQs) with options and a correct answer flag for the selected topic. This generation will incorporate a tool to help in deciding whether the questions it has selected, reflect multiple difficulty levels
- Loading Indicator: Displays a loader while the MCQs are being generated to let the user know that it might take some time.
- Question Display: Presents the questions one by one with 'next' and 'previous' navigation buttons.
- Progress Bar: Shows a progress bar to indicate the user's progress through the quiz.
- Score-Based Feedback: AI generates a personalized feedback message based on the user's score upon quiz completion.

## Style Guidelines:

- Primary color: Soft purple (#A093FF) to convey a sense of calm and focus, appropriate for learning.
- Background color: Very light lavender (#F5F4FF), visibly related to the purple.
- Accent color: Light blue (#7DD3FC) to provide contrast, but still within the analogous hues.
- Body and headline font: 'PT Sans', a humanist sans-serif.
- Use clean, modern icons for navigation and topic selection.
- Subtle animations for transitions between questions and feedback display.
